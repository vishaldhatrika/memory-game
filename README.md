# Meme-ory Game

 - Progressive Web App
 - Fun memory game based on tile matching, using memes as tile subjects
 - React + Typescript
 - Install on most devices with a browser
 - Play offline

[https://memorygame.vishaldhatrika.me](https://memorygame.vishaldhatrika.me)
